# homebrew-tap

Homebrew tap for onevcat tools.

## Install ca

```bash
brew install onevcat/tap/ca
```

## Install tang

```bash
brew install onevcat/tap/tang
```

## Install Prowl

```bash
brew install --cask onevcat/tap/prowl
```

## Bottles

Bottles are built and published automatically after a Chroma release for supported platforms.
If a bottle is not available, Homebrew will build from source.
